# Machine-Spam-Email-Classifier
•	Spam email detection is a common problem in machine learning. The goal of this project is to build supervised models to classify emails as Ham or Spam.
•	Three different models were trained and evaluated: Multinomial Naive Bayes, K-Nearest Neighbors (KNN), and Logistic Regression.
•	The project demonstrates the impact of algorithm choice and dataset balance on model performance.
